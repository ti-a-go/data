## Install and Run

Esse projeto utiliza modelos pré-treinados disponível na biblioteca Spacy.

Para baixar os modelos execute o script `spacy_download.sh`

Os modelos baixados:
- pt_core_news_lg
- es_dep_news_trf
- zh_core_web_sm
